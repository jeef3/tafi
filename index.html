What is Tafi?

Silly name really, but I have no idea what else to call this thing. It is basically an advanced formatted input. It allows you to restrict input to that which follows a structured format, selecting only from a list of known options. Perhaps where each option you select, decides a different path of available options for the rest of the input.

Could you show me an example?

Perhaps take phone numbers as an example. If you were to start typing a phone number, there is a finite list of country codes to choose from. Let's call up New Zealand: +64.

With +64 set as our country code, we know that the next part of the number must come from one of New Zealands subscriber toll dialling options.

If you had dialled up +1 for The United States, then there would be a completely different list of options to choose from.
